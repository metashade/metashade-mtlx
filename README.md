# metashade-mtlx (DEPRECATED)

> ⚠️ **This repository has been deprecated and archived.**

The functionality of `metashade-mtlx` has been consolidated into the main [metashade](https://github.com/metashade/metashade) repository as the [`metashade.mtlx`](https://github.com/metashade/metashade/tree/main/metashade/mtlx) package.

## Migration

If you were using `metashade-mtlx`, update your imports:

```python
# Old
from metashade_mtlx import generate

# New  
from metashade.mtlx import generate
```

Install the unified package:

```bash
pip install metashade
```

This repository remains available as a read-only archive for historical reference.
